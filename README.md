# 前言

2008年比特币诞生，原本只是一个密码学极客之间的玩物，没想到犹如打开的潘多拉盒子，慢慢席卷全球。在08 年以前还没有人能成功地研发出一个运行良好的数字货币出来，直到比特币问世;另外，区块链作为比特币的底层 技术，在此之前也是闻所未闻。那么区块链到底有什么魔力，让整个世界为之疯狂呢?
